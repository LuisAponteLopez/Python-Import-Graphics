

#juegocerdo_luisapontelopez.py 
Este es un programa donde el usuario va a jugar un juego donde se lanza un dado y la unica manera de perder es si el dado sale 1. 
Si sale 1 su puntuacion se hace a 0 y pierdes .Si sale  numeros >1 entonce se van sumando hasta que el usuario diga parar , para 
terminar el juego.El programa funciona a un 100% y proximamente viene con juegocerdo.01_luisapontelopez una nueva vercion donde se le 
estara implementado audio y anuncion.
